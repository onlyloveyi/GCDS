GitHub 部署助手

此工具可以帮助您轻松地将网站部署到 GitHub Pages。所有操作都在本地执行，不会发送您的个人信息到任何服务器。
步骤 1: 设置 GitHub 信息
您可以在 GitHub 的 Settings > Developer settings > Personal access tokens 创建令牌。 确保授予 "repo" 权限。
不建议使用密码认证，GitHub 可能会拒绝密码认证尝试。 出于安全考虑，请优先使用个人访问令牌 (PAT)。
步骤 2: 运行以下命令
打开终端，切换到您项目的根目录，然后依次运行以下命令： 
git remote add origin https://github.com/username/repo.git
git push -u origin main
步骤 3: 启用 GitHub Pages
  1. 访问您的 GitHub 仓库页面： https://github.com/username/repo
  2. 点击顶部的 "Settings" 选项卡
  3. 在左侧菜单中，点击 "Pages"
  4. 在 "Source" 部分，选择 "Deploy from a branch"
  5. 在 "Branch" 下拉菜单中，选择 "main"，然后点击 "Save"
  6. 等待几分钟，然后刷新页面，您的站点网址将显示在页面顶部
您的网站将部署在： https://username.github.io/repo/
